The application here that is build using python flask is a PDF Splitter. It extracts pages from the uploaded PDF in the specified page rage. 
The front end for this application which is a Flask service accepting input from the user through the webbrowser. The front end is python flask service with Jinja2 templating. Flask is a micro web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components where pre-existing third-party libraries provide common functions. Jinja2 templating is used to interact with the HTML templates and python code.
The backend is written in PyPDF2. PyPDF2 is a python library built as a PDF toolkit.
